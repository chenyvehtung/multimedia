三. 列表(list)

3.1 ul（unordered list无序列表）

格式：<ul>表项序列</ul>
说明：为表项集合，与上下段之间各有一空行。每个表项前有项目符，表项后换行；同级表项左对齐、嵌套的次级表项右缩进
显示效果：
    无序列表
  • 标题
    • 标题级别
    • 标题对齐
  • 段落
  • 字体
    • 字体大小
      • 英文字体
      • 中文字体
    • 字体种类
    • 字体颜色

3.2 ol（ordered list有序列表）

说明：格式、语法、说明等都似ul，只是项目符为数字或字母所组成的序号符
显示效果：
    有序列表
  1. 标题
    1. 标题级别
    2. 标题对齐
  2. 段落
  3. 字体
    1. 字体大小
      1. 英文字体
      2. 中文字体
    2. 字体种类
    3. 字体颜色

3.3 dl（definition list定义列表）

语法：
  • <dl> (dt | dd)+ </dl>
  • <dt> (行内元素)* </dt>
  • <dd> (流元素)*</dd>
说明：dt与dd元素的结束标签可省，定义项与定义描述一般成对出现
显示效果：
定义列表
dl
Definition List
dt
Definition Term
dd
Definition Description

四. 图片(image)

浏览器一般在网页中支持GIF/JPEG/PNG图片格式。图片在网页中可用作网页背景图、嵌入式插图和超链接的热点

4.1 背景图

格式：<body background=“图片文件地址URI”> 文档体内容</body>
说明：图片会像瓷砖一样重复排列，铺满网页窗口

4.2 插图

格式：<img src = uri alt = text …/>
说明：img元素将资源指定的图片嵌入文档中定义该元素的位置处。img为空标签
